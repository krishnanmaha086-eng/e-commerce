# ElectroHub – MERN E-commerce Website

ElectroHub is a full-stack e-commerce website built using the MERN stack.

## Features
- Product Listing
- Product Detail Page
- Category Filter
- Search with Suggestions
- Add to Cart
- Quantity Update
- Checkout Page
- User Login / Register
- User-wise Cart Storage
- Toast Notifications
- Responsive UI

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Bootstrap
- CSS Modules

## Project Structure
/backend → API + Database  
/frontend → React UI
